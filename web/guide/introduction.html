PhiChartSearch 介绍 ​

PhiChartSearch 是一个基于 Python 开发的 Phigros 官谱提取辅助工具，主要功能包括谱面文件解析、音频时长匹配和文件管理。

🎯 项目概述 ​

这个工具是为了解决从 Phigros 游戏文件中提取和整理官方谱面的一些需求而开发的：

  • 谱面文件解析：读取 Phigros 官方谱面 JSON 文件，提取基本信息
  • 音频时长匹配：根据谱面时长查找对应的官方音频文件
  • 文件管理：整理和管理提取的谱面和音频文件

✨ 基本功能 ​

🎵 音频匹配 ​

  • 读取 WAV 文件并计算实际时长
  • 根据目标时长进行简单的匹配度计算
  • 支持精确到 0.01 秒的时长显示
  • 记住上次使用的音频文件夹路径

📝 谱面分析 ​

  • 解析 Phigros 谱面 JSON 文件
  • 提取谱面名称、难度、BPM 等信息
  • 计算谱面物量（音符数量）
  • 估算谱面时长

🏗️ 工程管理 ​

  • 创建工程文件夹
  • 生成 8 位随机 Path
  • 创建 info.txt 配置文件
  • 可选的曲绘图片生成

🖥️ 界面设计 ​

  • 基于 tkinter 的简单图形界面
  • 窗口互斥设计，避免多窗口混乱
  • 基本的文件选择和操作功能

📋 系统要求 ​

运行环境 ​

  • Python 3.7+
  • Windows/macOS/Linux

依赖库 ​

tkinter - GUI界面
PIL (Pillow) - 图像处理
wave - 音频文件处理
json - 配置文件处理

🚀 快速开始 ​

1. 克隆项目 ​

bash
git clone https://github.com/catmcbe/PhiChartSearch
cd PhiChartSearch-main

2. 运行程序 ​

bash
python ChartAnalyzer.py

如果你不想配置环境，可前往releases下载已编译的exe文件 ​

3. 首次使用 ​

  1. 程序会自动检测是否存在未完成的工程
  2. 如果没有，会弹出工程创建向导
  3. 按照向导填写工程信息
  4. 开始谱面搜索和音频匹配

📖 使用指南 ​

创建新工程 ​

  1. 填写工程名称和难度（必填）
  2. 选择工程文件夹位置
  3. 可选：填写Composer和Charter信息
  4. 可选：启用自动曲绘生成
  5. 点击"创建工程"

谱面搜索 ​

  1. 选择谱面文件夹（TextAsset）
  2. 设置筛选条件：
    • 关键词（EZ、HD、IN、AT等）
    • 目标物量
    • 目标BPM
    • 目标音频长度
  3. 点击"开始筛选"
  4. 从结果中选择合适的谱面
  5. 点击"添加到工程"

音频匹配 ​

  1. 选择音频文件夹（包含WAV文件）
  2. 输入目标音频时长
  3. 点击"开始筛选"
  4. 从匹配结果中选择音频
  5. 点击"添加到工程"

🔧 配置说明 ​

程序配置 ​

程序会自动创建以下配置文件：

  • chart_analyzer_config.json：用户偏好设置
  • project_config.json：工程信息存储

配置文件结构 ​

json
{
  "last_folder": "D:\\TextAsset",
  "audio_folder": "D:\\Audio"
}

📊 技术架构 ​

核心模块 ​

  • ChartAnalyzer：主程序入口
  • Chart类：谱面数据模型
  • AudioFile类：音频文件模型
  • 工程管理模块：工程创建和管理

算法说明 ​

  • 匹配度计算：基于物量、BPM、时长的综合评分
  • 音频时长分析：使用wave库精确计算
  • 智能排序：按匹配度降序排列结果

🐛 故障排除 ​

常见问题 ​

Q: 程序启动时崩溃 A: 检查Python版本和依赖库是否正确安装

Q: 无法找到谱面文件 A: 确保谱面文件夹路径正确，文件格式为JSON

Q: 音频匹配失败 A: 检查音频文件是否为WAV格式，文件夹权限是否正确

Q: 工程创建失败 A: 确保工程文件夹有写入权限，路径不包含特殊字符

Q：BPM不确定？ A: 尝试不使用bpm筛选功能

Q：曲绘自动生成失败？ A: 确保字体Source Han Sans & Saira Hybrid-Regular #2934.ttf在程序所在目录

Q： 无法自动退出程序？经常报错？ A: 当前版本为测试版本，BUG很多，会逐渐完善

错误代码说明 ​

  • KeyError：谱面文件格式错误
  • FileNotFoundError：文件路径不存在
  • PermissionError：文件权限不足
