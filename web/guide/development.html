开发思路与代码剖析 ​

本文档介绍 PhiChartSearch 的开发思路和核心代码实现。

💡 开发背景 ​

在提取 Phigros 官方谱面时，经常需要：

  1. 从游戏文件中筛选和提取官方谱面
  2. 为提取的谱面匹配对应的官方音频文件
  3. 整理和管理提取的文件

手动处理这些工作既耗时又容易出错，因此开发了这个辅助工具。

🏗️ 整体架构 ​

技术栈选择 ​

  • Python 3.7+：主要开发语言
  • tkinter：GUI界面开发（Python内置，无需额外安装）
  • PIL (Pillow)：图像处理，用于曲绘生成
  • wave：音频文件处理，计算音频时长
  • json：配置文件和谱面文件解析

项目结构 ​

PhiChartSearch/
├── ChartAnalyzer.py      # 主程序入口
├── Chart类.py           # 谱面数据模型
├── AudioFile类.py       # 音频文件模型
├── 工程管理模块.py       # 工程创建和管理
├── chart_analyzer_config.json  # 用户配置
├── project_config.json         # 工程配置
└── 字体文件.ttf               # 曲绘生成字体

🔧 核心功能实现 ​

1. 谱面解析模块 ​

设计思路 ​

Phigros 谱面文件是 JSON 格式，包含音符数据、BPM 信息等。需要解析这些文件并提取关键信息。

代码实现 ​

python
import json
import os

class Chart:
    def __init__(self, file_path):
        self.file_path = file_path
        self.file_name = os.path.basename(file_path)
        self.name = ""
        self.difficulty = ""
        self.bpm = 0
        self.duration = 0
        self.note_count = 0
        
        self.parse_chart()
    
    def parse_chart(self):
        """解析谱面文件"""
        try:
            with open(self.file_path, 'r', encoding='utf-8') as f:
                data = json.load(f)
            
            # 提取基本信息
            self.extract_basic_info(data)
            
            # 计算 BPM
            self.calculate_bpm(data)
            
            # 计算时长和物量
            self.calculate_duration_and_notes(data)
            
        except Exception as e:
            print(f"解析谱面文件失败 {self.file_path}: {e}")
    
    def extract_basic_info(self, data):
        """从文件名和内容中提取基本信息"""
        # 从文件名提取难度
        if "EZ" in self.file_name:
            self.difficulty = "EZ"
        elif "HD" in self.file_name:
            self.difficulty = "HD"
        elif "IN" in self.file_name:
            self.difficulty = "IN"
        elif "AT" in self.file_name:
            self.difficulty = "AT"
        
        # 提取谱面名称
        self.name = data.get("name", "")
    
    def calculate_bpm(self, data):
        """计算 BPM"""
        bpm_list = data.get("bpmList", [])
        if bpm_list:
            # 取第一个 BPM 作为主要 BPM
            self.bpm = bpm_list[0].get("bpm", 0)
    
    def calculate_duration_and_notes(self, data):
        """计算时长和物量"""
        notes = data.get("notes", [])
        if not notes:
            return
        
        # 计算物量
        self.note_count = len(notes)
        
        # 计算时长（基于最后一个音符的时间）
        max_time = 0
        for note in notes:
            note_time = note.get("time", 0)
            if note_time > max_time:
                max_time = note_time
        
        # 转换为秒
        self.duration = max_time / 1000.0  # 假设时间单位是毫秒

2. 音频匹配模块 ​

设计思路 ​

使用 Python 的 wave 库读取 WAV 文件，计算实际时长，然后与目标时长进行比较计算匹配度。

代码实现 ​

python
import wave
import os

class AudioFile:
    def __init__(self, file_path):
        self.file_path = file_path
        self.file_name = os.path.basename(file_path)
        self.duration = 0
        self.match_score = 0
        
        self.calculate_duration()
    
    def calculate_duration(self):
        """计算音频时长"""
        try:
            with wave.open(self.file_path, 'rb') as wav_file:
                # 获取音频参数
                frames = wav_file.getnframes()
                sample_rate = wav_file.getframerate()
                
                # 计算时长（秒）
                self.duration = frames / float(sample_rate)
                
        except Exception as e:
            print(f"读取音频文件失败 {self.file_path}: {e}")
            self.duration = 0
    
    def calculate_match_score(self, target_duration):
        """计算匹配度"""
        if self.duration <= 0 or target_duration <= 0:
            return 0
        
        # 计算时间差
        time_diff = abs(self.duration - target_duration)
        
        # 匹配度计算：时间差越小，匹配度越高
        # 使用简单的线性公式
        if time_diff <= 1.0:  # 1秒内差异
            self.match_score = 100 - (time_diff * 50)
        else:
            self.match_score = max(0, 50 - (time_diff * 10))
        
        return self.match_score

def find_matching_audio_files(audio_folder, target_duration, min_score=50):
    """查找匹配的音频文件"""
    matching_files = []
    
    if not os.path.exists(audio_folder):
        return matching_files
    
    for file_name in os.listdir(audio_folder):
        if file_name.lower().endswith('.wav'):
            file_path = os.path.join(audio_folder, file_name)
            audio_file = AudioFile(file_path)
            
            # 计算匹配度
            score = audio_file.calculate_match_score(target_duration)
            
            if score >= min_score:
                matching_files.append(audio_file)
    
    # 按匹配度排序
    matching_files.sort(key=lambda x: x.match_score, reverse=True)
    
    return matching_files

3. 工程管理模块 ​

设计思路 ​

创建工程文件夹，生成必要的配置文件，包括 info.txt 和 project_config.json。

代码实现 ​

python
import json
import os
import random
import string
from PIL import Image, ImageDraw, ImageFont

class ProjectManager:
    def __init__(self):
        self.config_file = "project_config.json"
        self.projects = self.load_projects()
    
    def create_project(self, name, difficulty, folder_path, composer="", charter=""):
        """创建新工程"""
        # 生成唯一的 8 位 Path
        path = self.generate_path()
        
        # 创建工程文件夹
        project_folder = os.path.join(folder_path, f"{name}_{difficulty}")
        os.makedirs(project_folder, exist_ok=True)
        
        # 创建工程信息
        project_info = {
            "name": name,
            "difficulty": difficulty,
            "path": path,
            "composer": composer,
            "charter": charter,
            "folder": project_folder,
            "created_time": self.get_current_time()
        }
        
        # 保存工程信息
        self.projects[path] = project_info
        self.save_projects()
        
        # 生成工程文件
        self.generate_project_files(project_info)
        
        return project_info
    
    def generate_path(self):
        """生成8位随机Path"""
        while True:
            # 生成8位字母数字组合
            path = ''.join(random.choices(string.ascii_uppercase + string.digits, k=8))
            
            # 检查是否已存在
            if path not in self.projects:
                return path
    
    def generate_project_files(self, project_info):
        """生成工程文件"""
        folder = project_info["folder"]
        
        # 生成 info.txt
        self.generate_info_txt(project_info)
        
        # 生成曲绘（可选）
        self.generate_illustration(project_info)
    
    def generate_info_txt(self, project_info):
        """生成 info.txt 文件"""
        folder = project_info["folder"]
        info_path = os.path.join(folder, "info.txt")
        
        content = f"""[基本信息]
名称={project_info["name"]}
难度={project_info["difficulty"]}
Composer={project_info["composer"]}
Charter={project_info["charter"]}
Path={project_info["path"]}
"""
        
        with open(info_path, 'w', encoding='utf-8') as f:
            f.write(content)
    
    def generate_illustration(self, project_info):
        """生成曲绘图片"""
        try:
            font_path = "Source Han Sans & Saira Hybrid-Regular #2934.ttf"
            if not os.path.exists(font_path):
                print("字体文件不存在，跳过曲绘生成")
                return
            
            # 创建图片
            width, height = 1024, 1024
            image = Image.new('RGB', (width, height), color='#2c3e50')
            draw = ImageDraw.Draw(image)
            
            # 加载字体
            font = ImageFont.truetype(font_path, 80)
            
            # 绘制文字
            text = project_info["name"]
            bbox = draw.textbbox((0, 0), text, font=font)
            text_width = bbox[2] - bbox[0]
            text_height = bbox[3] - bbox[1]
            
            x = (width - text_width) // 2
            y = (height - text_height) // 2
            
            draw.text((x, y), text, font=font, fill='white')
            
            # 保存图片
            img_path = os.path.join(project_info["folder"], "illustration.jpg")
            image.save(img_path)
            
        except Exception as e:
            print(f"生成曲绘失败: {e}")

4. GUI界面模块 ​

设计思路 ​

使用 tkinter 创建简单的图形界面，采用窗口互斥设计，避免多个窗口同时显示造成混乱。

代码实现 ​

python
import tkinter as tk
from tkinter import ttk, filedialog, messagebox

        self.root = tk.Tk()
        self.root.title("PhiChartSearch")
        self.root.geometry("800x600")
        
        # 当前打开的窗口
        self.current_window = None

        self.config = self.load_config()

        self.create_main_interface()

        menubar = tk.Menu(self.root)
        self.root.config(menu=menubar)
        
        # 文件菜单
        file_menu = tk.Menu(menubar, tearoff=0)
        menubar.add_cascade(label="文件", menu=file_menu)
        file_menu.add_command(label="新建工程", command=self.create_project)
        file_menu.add_command(label="打开工程", command=self.open_project)

🐛 遇到的问题和解决方案 ​

1. 音频文件读取问题 ​

问题：某些 WAV 文件无法正常读取 解决：添加异常处理，跳过损坏的文件

2. 中文路径问题 ​

问题：包含中文的路径可能导致文件操作失败 解决：使用正确的编码格式，建议用户使用英文路径

3. 内存占用问题 ​

问题：处理大量文件时内存占用过高 解决：优化算法，避免一次性加载所有文件

4. GUI 响应性问题 ​

问题：长时间操作导致界面卡顿 解决：使用多线程处理耗时操作

📈 性能优化 ​

1. 文件缓存 ​

  • 缓存已解析的谱面信息
  • 避免重复解析相同文件

2. 异步处理 ​

  • 使用线程处理文件操作
  • 保持界面响应性

3. 内存管理 ​

  • 及时释放不需要的对象
  • 控制同时处理的文件数量
