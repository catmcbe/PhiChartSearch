🚨 常见问题 ​

程序启动问题 ​

Q: 程序无法启动，双击没反应 ​

可能原因：

  • Python 版本不兼容
  • 依赖库未正确安装
  • 系统权限不足

解决方案：

  1. 检查 Python 版本：

    bash
    python --version

    确保版本为 3.7 或更高

  2. 安装依赖库：

    bash
    pip install pillow

  3. 以管理员身份运行程序

  4. 检查防病毒软件是否误报

Q: 启动时出现命令行窗口闪退 ​

可能原因：

  • 程序运行时出错
  • 缺少必要的文件
  • 路径包含特殊字符

解决方案：

  1. 通过命令行启动查看错误信息：

    bash
    python ChartAnalyzer.py

  2. 检查程序目录是否完整：

    • ChartAnalyzer.py
    • 字体文件（如果使用曲绘生成）
    • 其他配置文件

  3. 将程序移到简单路径（如 C:\PhiChartSearch）

Q: 提示 "tkinter 模块未找到" ​

可能原因：

  • Python 安装时未包含 tkinter
  • 使用了精简版 Python

解决方案：

  1. 重新安装完整版 Python
  2. 或安装 tkinter：
    bash
    # Ubuntu/Debian
sudo apt-get install python3-tk

# Fedora
sudo dnf install python3-tkinter

# macOS (如果使用 Homebrew)
brew install python-tk
