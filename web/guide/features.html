🎵 音频匹配功能 ​

功能概述 ​

音频匹配功能是 PhiChartSearch 的核心功能之一，它能够根据谱面的音频时长智能匹配最合适的 WAV 音频文件。

工作原理 ​

音频时长分析 ​

  • 使用 Python 的 wave 库精确计算音频文件时长
  • 支持精确到小数点后两位的时长计算
  • 自动跳过非 WAV 格式的音频文件

智能匹配算法 ​

python
# 匹配度计算公式
匹配度 = 100 - abs(目标时长 - 实际时长) * 系数

使用步骤 ​

  1. 选择音频文件夹

    • 点击"选择音频文件夹"按钮
    • 浏览并选择包含 WAV 文件的文件夹
    • 程序会自动记忆该路径

  2. 输入目标时长

    • 在输入框中输入目标音频时长
    • 支持格式：
      • 120（120秒）
      • 120.5（120.5秒）
      • 2:00（2分钟，会自动转换为秒）

  3. 开始筛选

    • 点击"开始筛选"按钮
    • 程序会扫描文件夹中的所有 WAV 文件
    • 计算每个文件的匹配度并排序

  4. 查看结果

    • 结果按匹配度从高到低排列
    • 显示文件名、时长、匹配度等信息
    • 匹配度用百分比和颜色标识

  5. 添加到工程

    • 选择最合适的音频文件
    • 点击"添加到工程"按钮
    • 文件会被复制到工程文件夹

高级选项 ​

匹配度阈值 ​

  • 默认显示匹配度 > 50% 的结果
  • 可以在设置中调整阈值

文件过滤 ​

  • 支持按文件名关键词过滤
  • 支持按文件大小过滤

🏗️ 工程管理系统 ​

功能概述 ​

工程管理系统提供完整的谱面工程创建、编辑和管理功能，支持多工程并行管理。

工程创建向导 ​

基本信息设置 ​

  • 工程名称：谱面的名称，必填
  • 难度：从下拉列表选择（EZ、HD、IN、AT、SP等）
  • 工程文件夹：选择工程保存位置

可选信息 ​

  • Composer：作曲家名称
  • Charter：谱师名称
  • Illustrator：曲绘作者
  • 自动曲绘生成：是否自动生成曲绘图片

Path 生成 ​

  • 自动生成8位随机字符串作为工程Path
  • 格式：XXXXXXXX（8位字母数字组合）
  • 确保每个工程有唯一标识

工程文件结构 ​

工程文件夹/
├── info.txt          # 工程信息文件
├── chart.json        # 谱面文件
├── audio.wav         # 音频文件
├── illustration.jpg  # 曲绘图片（可选）
└── project_config.json # 工程配置文件

工程配置 ​

info.txt 格式 ​

[基本信息]
名称=工程名称
难度=HD
Composer=作曲家
Charter=谱师
Illustrator=画师
Path=ABC12345

project_config.json 格式 ​

json
{
  "project_name": "工程名称",
  "difficulty": "HD",
  "path": "ABC12345",
  "composer": "作曲家",
  "charter": "谱师",
  "illustrator": "画师",
  "chart_file": "chart.json",
  "audio_file": "audio.wav",
  "illustration_file": "illustration.jpg",
  "created_time": "2024-01-01 12:00:00",
  "last_modified": "2024-01-01 12:30:00"
}

🔧 谱面分析功能 ​

功能概述 ​

谱面分析功能能够智能解析 Phigros 谱面文件，提取关键信息用于筛选和匹配。

支持的谱面格式 ​

  • Phigros 官方谱面格式（JSON）
  • 标准化的自定义谱面格式
  • 兼容大部分第三方谱面工具生成的格式

提取的信息 ​

基本信息提取 ​

  • 谱面名称：从文件名或内容中提取
  • 难度：从文件名或内容中识别
  • BPM：从谱面数据中计算得出
  • 时长：根据最后一个音符的时间计算

物量统计 ​

  • 总物量：所有音符的总数
  • 分类统计：
    • 单击音符数量
    • 长按音符数量
    • 滑动音符数量
    • 特殊音符数量

难度分析 ​

  • 密度分析：计算单位时间内的音符密度
  • 复杂度评估：基于音符类型和分布评估复杂度

筛选功能 ​

关键词筛选 ​

  • 支持多个关键词：EZ,HD,IN
  • 支持正则表达式：EZ|HD|IN
  • 支持排除关键词：-EZ（排除EZ难度）

数值筛选 ​

  • 物量筛选：
    • 精确匹配：300
    • 范围匹配：200-400
    • 大于/小于：>200、<500
  • BPM筛选：
    • 支持小数：120.5
    • 范围匹配：100-150
  • 时长筛选：
    • 秒数：120
    • 分秒格式：2:00
    • 范围匹配：100-180

组合筛选 ​

  • 支持多个条件同时筛选
  • 条件之间为"与"关系
  • 可以随时添加或删除筛选条件

🎨 用户界面功能 ​

界面设计理念 ​

  • 简洁直观：界面布局清晰，操作逻辑简单
  • 窗口互斥：主要功能窗口互斥显示，避免混乱
  • 实时反馈：操作结果实时显示，提供详细状态信息

主要界面组件 ​

主界面 ​

  • 菜单栏：文件、编辑、视图、帮助
  • 工具栏：常用功能快捷按钮
  • 状态栏：显示当前状态和提示信息
  • 工作区：主要功能区域

工程创建界面 ​

  • 表单区域：输入工程信息
  • 预览区域：实时预览工程信息
  • 进度指示器：显示创建进度
  • 操作按钮：确认、取消、帮助

谱面搜索界面 ​

  • 筛选条件区：设置搜索条件
  • 结果列表区：显示搜索结果
  • 详细信息区：显示选中谱面的详细信息
  • 操作按钮区：添加、刷新、导出等操作

音频匹配界面 ​

  • 文件夹选择区：选择音频文件夹
  • 时长输入区：输入目标时长
  • 匹配结果区：显示匹配结果
  • 文件预览区：预览选中音频

交互特性 ​

拖拽支持 ​

  • 支持拖拽文件夹到程序窗口
  • 支持拖拽文件到列表中

快捷键支持 ​

  • Ctrl+O：打开文件夹
  • Ctrl+F：搜索
  • Ctrl+N：新建工程
  • F5：刷新
  • Esc：取消当前操作

上下文菜单 ​

  • 右键点击文件显示相关操作
  • 支持批量操作
  • 快速访问常用功能

⚡ 自动化功能 ​

自动曲绘生成 ​

生成原理 ​

  • 使用工程名称生成曲绘
  • 支持多种字体和样式
  • 自动调整字体大小和位置

生成选项 ​

  • 字体选择：支持多种字体
  • 颜色方案：预设多种配色方案
  • 布局样式：居中、左对齐、右对齐
  • 背景样式：纯色、渐变、图案

字体要求 ​

  • 需要字体文件：Source Han Sans & Saira Hybrid-Regular #2934.ttf
  • 字体文件应放在程序目录下
  • 支持中英文显示

自动文件生成 ​

info.txt 自动生成 ​

  • 根据工程信息自动生成
  • 格式符合 Phigros 标准
  • 支持自定义模板

配置文件自动生成 ​

  • 工程配置自动保存
  • 用户偏好自动记录
  • 历史记录自动维护

批量操作 ​

批量添加 ​

  • 支持批量添加谱面到工程
  • 支持批量处理音频文件
  • 自动去重和验证

批量导出 ​

  • 支持批量导出工程信息
  • 支持批量生成配置文件
  • 支持批量格式转换

📊 数据可视化 ​

匹配度可视化 ​

  • 使用颜色条显示匹配度
  • 绿色：高匹配度（>80%）
  • 黄色：中等匹配度（50-80%）
  • 红色：低匹配度（<50%）

统计图表 ​

  • 物量分布图
  • BPM分布图
  • 时长分布图
  • 难度分布图

进度指示 ​

  • 实时进度条
  • 百分比显示
  • 剩余时间估算
  • 操作状态提示
