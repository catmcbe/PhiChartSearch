快速开始 ​

本指南将帮助您快速上手 PhiChartSearch，从安装到第一次使用。

📦 安装 ​

方式一：下载可执行文件（推荐） ​

  1. 前往 GitHub Releases
  2. 下载最新版本的 PhiChartSearch.exe
  3. 双击运行即可，无需额外配置

方式二：从源码运行 ​

系统要求 ​

  • Python 3.7 或更高版本
  • Windows/macOS/Linux 操作系统

安装步骤 ​

  1. 克隆项目

    bash
    git clone https://github.com/catmcbe/PhiChartSearch
cd PhiChartSearch-main

  2. 安装依赖

    bash
    pip install pillow

  3. 运行程序

    bash
    python ChartAnalyzer.py

🚀 首次使用 ​

启动程序 ​

双击 PhiChartSearch.exe 或运行 python ChartAnalyzer.py 启动程序。

创建第一个工程 ​

  1. 工程创建向导

    • 程序启动后会自动检测是否存在未完成的工程
    • 如果没有，会弹出工程创建向导

  2. 填写基本信息

    • 工程名称：输入您的谱面名称（必填）
    • 难度：选择谱面难度（必填）
    • 工程文件夹：选择保存工程的位置

  3. 可选信息

    • Composer：作曲家名称
    • Charter：谱师名称
    • 自动曲绘生成：是否自动生成曲绘图片

  4. 创建工程

    • 点击"创建工程"按钮完成创建
    • 程序会自动生成8位随机Path

📋 基本使用流程 ​

步骤 1：搜索谱面 ​

  1. 选择谱面文件夹

    • 点击"选择文件夹"按钮
    • 导航到包含 Phigros 谱面的 TextAsset 文件夹

  2. 设置筛选条件

    • 关键词：输入难度标识（如 EZ、HD、IN、AT等）
    • 目标物量：设置期望的物量范围
    • 目标BPM：设置期望的BPM范围
    • 目标音频长度：设置期望的音频时长

  3. 开始搜索

    • 点击"开始筛选"按钮
    • 等待搜索完成，查看匹配结果

  4. 选择谱面

    • 从结果列表中选择合适的谱面
    • 点击"添加到工程"按钮

步骤 2：匹配音频 ​

  1. 选择音频文件夹

    • 点击"选择音频文件夹"按钮
    • 选择包含 WAV 音频文件的文件夹

  2. 输入目标时长

    • 根据谱面信息输入目标音频时长
    • 支持精确到小数点后两位

  3. 开始匹配

    • 点击"开始筛选"按钮
    • 查看匹配结果和匹配度

  4. 选择音频

    • 从结果中选择最匹配的音频文件
    • 点击"添加到工程"按钮

步骤 3：完成工程 ​

  1. 检查工程信息

    • 确认谱面和音频都已添加
    • 检查工程配置是否正确

  2. 生成文件

    • 程序会自动生成必要的文件：
      • info.txt：工程信息文件
      • 曲绘图片（如果启用自动生成）
      • 其他配置文件

  3. 保存工程

    • 工程信息会自动保存到配置文件
    • 下次启动时可以继续编辑

⚙️ 高级功能 ​

自定义筛选 ​

物量筛选 ​

  • 支持范围筛选：100-500
  • 支持精确匹配：300
  • 支持大于/小于：>200、<400

BPM筛选 ​

  • 支持小数点：120.5
  • 支持范围筛选：100-200
  • 支持模糊匹配：如果BPM不确定，可以留空

时长筛选 ​

  • 精确到秒：120
  • 精确到0.01秒：120.35
  • 支持范围筛选：100-150

音频文件夹记忆 ​

程序会记住您上次使用的音频文件夹路径，下次使用时自动选择。

匹配度算法 ​

匹配度基于以下因素计算：

  • 物量匹配度：30%
  • BPM匹配度：30%
  • 时长匹配度：40%

结果按匹配度从高到低排序。

🔧 故障排除 ​

常见问题解决 ​

程序无法启动 ​

  1. 检查 Python 版本是否为 3.7+
  2. 确保安装了所需的依赖库
  3. 检查是否有足够的系统权限

找不到谱面文件 ​

  1. 确保选择了正确的 TextAsset 文件夹
  2. 检查文件夹中是否包含 JSON 格式的谱面文件
  3. 确保文件没有被其他程序占用

音频匹配失败 ​

  1. 确保音频文件为 WAV 格式
  2. 检查音频文件是否损坏
  3. 确保文件夹有读取权限

工程创建失败 ​

  1. 确保目标文件夹有写入权限
  2. 检查路径是否包含特殊字符
  3. 确保磁盘空间充足

性能优化建议 ​

  1. 大型文件夹处理

    • 对于包含大量文件的文件夹，首次扫描可能需要较长时间
    • 建议将谱面和音频文件分类存放

  2. 内存使用

    • 程序会缓存搜索结果以提高性能
    • 如果内存不足，可以定期重启程序

  3. 文件路径

    • 避免使用过深的文件夹路径
    • 避免使用中文路径（在某些系统上可能出现问题）
